Introduction

An introduction into Group Over-Drive’s W03_560 Test Plan

Test Plan

This test plan describes what testing was done on the W03_560 Machine Assembler. The plan was designed in order to fix all errors and meet all of the W03_560 Machine Assembler’s specifications and requirements. This documentation includes a list of different categories of extreme cases and errors for which source code was tested.

Our Test Plan consists of two sections; the first section tests specific errors in the input program, the second one tests the robustness and correctness of specific functions and operations
