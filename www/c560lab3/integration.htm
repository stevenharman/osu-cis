W03-560 Machine
User's Integration Guide

Overview

This is a User's Guide for the integration of Group OverDrive's 3 main components for the operation of the W03-560 Machine. These components are the Group OverDrive Modified Assembler, Linking Loader, and Simulator, respectively. These components can be used in tandem to simulate the operation of the W03-560 Machine. The general order of execution for simulation of a program is:

  1. Assemble all source files (one at a time) using the Group OverDrive Modified Assembler
  2. Link/Load all assembled files using the Group OverDrive Linking Loader
  3. Execute the linked file using the Group OverDrive Simulator

Group OverDrive Modified Assembler

The Modified Assembler is the first component which will be used in creating an executable program. It is used to convert source code (text-format programs) into a format that can be understood by the Loader. It is important to note that programs assembled using Group OverDrive's old Assembler will NOT be compatible with the Linking Loader. The User's Guide for the Modified Assembler can be found here.

Group OverDrive Linking Loader

The Linking Loader takes multiple files which have been assembled by the Group OverDrive Modified Assembler and converts them into one file which is usable by the Group OverDrive Simulator. The User's Guide for the Linking Loader can be found here.

Group OverDrive Simulator

The Simulator executes the assembled, linked, and loaded file that is created by the Linking Loader. The User's Guide for the Simulator can be found here.
