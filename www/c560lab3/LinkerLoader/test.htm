Introduction

An introduction into Group Over-Drive’s W03_560 Test Plan

Test Plan

This test plan describes what testing was done on the W03_560 Machine Linker/Loader. The plan was designed in order to fix all errors and meet all of the W03_560 Machine Linker/Loader specifications and requirements. This documentation includes a list of different categories of extreme cases and errors for which source code was tested.

Our Test Plan consists of two sections; the first section tests specific errors in the input program, the second one tests the robustness and correctness of specific functions and operations. The following 3 assembly language programs were used for testing. The following programs are uncompiled but were compiled before each was used.

File Program.asm = obj1

Main ORI
EXT Disp1,V
ENT Start
EXT X
;
Start BRS 3,Disp1 ;Display 6..0
LD 1,V ;r1 <- M[V]
ST 1,X ;M[X] <- r1
BRS 3,Disp1 ;Display 2..0
BR 0,0 ;halt
END

File Subr.asm = obj2

Mesg ORI
ENT Disp1,X
TOUT EQU 3
NOUT EQU 2
;
Txt CCD Va
CCD lu
CCD e
CCD =
x NMD 6
SaveR Res 1
;
Disp1 ST 0,SaveR ;m[SaveR] <- r0
LD 0,X ;r0 <- M[x]
BRN 0,Done ;if (r0 < 0) goto Done
Loop IO TOUT,Txt ;Display text "Value = "
IO NOUT,X ;Display value in M[X]
SUB 0,=1 ;r0--
ST 0,x ;m[x] <- r0
BRN 0,Done ;if (r0 < 0) goto Done
BR 3,Loop ;goto Loop
Done LD 0,SaveR ;r0 <- M[SaveR]
BR 3,0(3) ;return
END

File Val.asm = obj3

Data ORI
EXT X
ENT V
V NMD 2
TOUT BR 0,0
Done BR 3,X
LD 1,=1
END Done
