Programmer's Guide: Draft

Main( ): Will be responsible for the UI
 
Machine Class: Will be responsible for loading, executing, and controlling the ‘Machine’
Functions & Procedures:
    int Text_To_Hex(char [ ] );
    void Simulate_This( char* fname);
Processor Class: Will be responsible for emulating the PC and R[0]-R[3]
  • PC: emulated by a 19-bit integer (even though the PC is 20-bits)
  • R[0]-R[4]: emulated by 4 20-bit integers
    		•
    Functions & Procedures:

      void Increment_PC( );
      void Set_PC(int n);
      int Get_PC( );
      void Load_Register( int reg_num, int reg_value);
      int Get_Register( int reg_num);
      void Put_Regs();
    Memory Class: Will be responsible for emulation Memory
  • Memory: emulated my array of 256 integers in decimal format
    		•
    Functions & Procedures:
      void Load_Memory(int mem_location, int mem_value);
      int Get_Memory( int mem_location);
      void Put_Mem();
    Error Class: Will be responsible for reporting errors that occur during Runtime
    Functions & Procedures:
      void Report_Error(err_num);
